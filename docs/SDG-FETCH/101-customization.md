# Customization

SDG-FETCH discovers logos and presets by scanning subdirectories of `~/.local/fetch/logos/` and `~/.local/fetch/conf/`. Any file in any subdirectory appears automatically in the picker, organized by its category folder.

## Adding your own logos

Create your own category and place ASCII art files there:

```sh
mkdir -p ~/.local/fetch/logos/my
# Place ASCII art files in ~/.local/fetch/logos/my/
```

Files in `~/.local/fetch/logos/my/` appear in the logo picker as `my/<filename>`.

To convert an image (PNG, JPG, JPEG, WEBP) to ASCII art:

```sh
sdgfetch convert mylogo.png
```

This outputs to `~/.local/fetch/logos/convert/mylogo`. You can then move it to your own category:

```sh
mv ~/.local/fetch/logos/convert/mylogo ~/.local/fetch/logos/my/
```

The `sdgfetch convert` command uses `jp2a` with `--height=22 --colors --background=dark`.

## Adding your own presets

Create your own category and place fastfetch JSONC preset files there:

```sh
mkdir -p ~/.local/fetch/conf/my
# Place *.jsonc files in ~/.local/fetch/conf/my/
```

Presets in `~/.local/fetch/conf/my/` appear in the preset picker as `my/<filename>.jsonc`.

## Where files live

| Path | Purpose |
|------|---------|
| `~/.local/fetch/logos/<category>/` | Logo files organized by category |
| `~/.local/fetch/conf/<category>/` | Preset files organized by category |
| `~/.local/fetch/logos/convert/` | Auto-generated by `sdgfetch convert` |
| `~/.config/sdgfetch.state` | Your current logo and preset selection |
