# Logo Library

SDG-FETCH ships with ASCII art logos in `~/.local/fetch/logos/SDG-FETCH/`. Each file is a text file containing ANSI-colored ASCII art generated by `jp2a`.

## Categories

### Distros
`archlinux`, `archbtw`, `cachy`, `nyarch`

### Gaming Brands
`ROG`, `alienware`, `aorus`, `msi`, `msi-text`, `razer`, `omen`, `acerpredator`

### Laptop Brands
`thinkpad`, `dell`, `hp`, `surface`

### Tech Culture
`LTT`, `WAN`, `ibm`, `apple`

### Memes & Culture
`cheese`, `dank`, `ganja`, `nobitches`, `snoop`, `warbooty`, `pusheen`, `noodle`, `mango`

### Pride
`pride`, `trans-pride`

### Themes & Fun
`destiny`, `gear`, `hypr`, `hypr-alt`, `rose`, `triangle`, `warframe`

### Windows Tributes
`win10`, `win11`, `win7`, `winold`

### Special Modes
`distro`, `distro-themed`, `none`

Special modes are handled internally by `fetch-cli.sh`:

- **`none`** — Runs fastfetch with `-l none`, no logo displayed
- **`distro`** — Uses fastfetch's built-in distro detection for the logo
- **`distro-themed`** — Uses fastfetch's built-in distro logo with themed ANSI color overrides

To add new logos, see the [Customization](101-customization.md) guide.
