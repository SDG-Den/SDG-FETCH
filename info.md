Package Name: sdg-fetch
Descriptive Name: SDG System Fetch
Source: https://github.com/SDG-Den/SDG-FETCH
Maintainer: SDGDen
Version:0.2.1

Dependencies: 
fastfetch, jp2a, fzf, bat

Description: 
Wrapper and configuration system around fastfetch. Provides an interactive logo picker (45 ASCII art logos) and style picker (49 display presets) with persistent state. Commands: sdgfetch (display system info), sdgfetch-conf (interactive configurator).
